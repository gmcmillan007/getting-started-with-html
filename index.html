What I learned this week

Main Concepts

What is the Web? The Web is all the HTML documents that exist in the largest network, which is a series of servers.

What is HTML? HTML stands for Hyper Text Markup Language. It is the language browsers interpret as what we see as pages in the Internet. At the end it is a programming language. There are over 20 billion HTML pages on the Internet.

What are URLs? URL stands for Uniform Resource Locator. It is the standard to request information and locate HTML pages on the Internet. A URL is sent to the server that stores the page requested. The server sends back the right page to be interpreted by the browser on your computer.

What is HTTP? It stands for Hyper Text Transfer Protocol, which is the protocol, or series of standardize instructions that the Internet uses to link HTML pages.

Web Applications were briefly mentioned as the goal for this course.

How does the Internet Works?

User sends a request from a device

The URL requested goes to the Internet

Through the HTTP protocol it is located in a server

The server sends the requested URL through HTTP

The URL travels through the Internet

The user’s browser interprets the HTML Page for the user to read/see/listen.

What is a Tag? A Tag is an HTML label describing the content of an HTML page. Tags are always contained within a less than and a more than brackets. And in most cases are used in pairs, an opening tag “(tag)” and a closing tag “(/tag)".

What is a Void Tag? A Void Tag has no content and therefore does not require a closing tag. Examples of a void tag are the bracket tag “(br), and image tag “(img)”.

What is an Inline Tag? Inline Tags are simple line tags. That is, they are not contained in boxes. Examples are: the (br), (img), (em), (b) tags, etc.

What is an Block Tag? Block Tags are always contained in boxes (although not always seen as such through their representation of the content. Examples are: The paragraph tag “(p)(/p)”, the anchor tag “(a)(/a)”, or the division tag “(div)(/div).

What is an Element? An Element is everything that is contained between an opening and a closing tag.

What is an Attribute? Attributes are properties assigned to the content in a tag, such as color, font, size, location, border size, etc.
